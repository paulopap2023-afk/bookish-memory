# ScriptFiscal PDV 2026

Sistema completo de emissão de **NFe e NFCe** desenvolvido em Scriptcase.

## Funcionalidades

- Emissão de NFe
- Emissão de NFCe
- PDV completo
- Controle de caixa
- Financeiro integrado
- Importação automática de XML
- Impressão ESC/POS
- QRCode NFCe

## Tecnologias

- Scriptcase
- PHP
- PostgreSQL
- XML SEFAZ
